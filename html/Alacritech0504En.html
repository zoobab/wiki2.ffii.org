Microsoft ordered to withdraw patented features from Longhorn

-> [ deutsch | français ] [ Patents in Action | Microsoft | News ]

14. April 2005 -- Microsoft was ordered by a US court to remove features from the upcoming version of its Windows operating system. The patent in question broadly claims all processes that delegate processing work in a network environment. The same patent is pending at the European Patent Office.
