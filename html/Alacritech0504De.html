Microsoft muss WIN XP-Nachfolger Longhorn abspecken

-> [ english | français ] [ Patente in Aktion | Microsoft | Neues ]

14. April 2005 -- Der Firma Microsoft wurde von einem Gericht per einstweilige Verfügung befohlen, Funktionen aus ihrem Betriebssystem Windows herauszunehmen. Die Auslieferung von Longhorn könnte sich verzögern. Microsoft verletzt einen breiten Patentanspruch der Firma Alacritech. Alacritech hatte sich zuvor als Kooperationspartner angeboten.
